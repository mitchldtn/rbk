# Navigation

## Project browser
- `↑↓` or `j/k` to move between projects
- `enter` to open a project
- `n` to create a new project
- `q` to quit

## Notes list
- `↑↓` or `j/k` to move between notes
- `enter` to open note in terminal
- `→` to browse note content and code blocks
- `←` or `esc` to go back

## In the terminal
- `ctrl+n` toggle the notes panel open/closed
- `ctrl+w` move focus between terminal and notes panel
- `ctrl+b` open note browser to switch notes
- `esc` return focus to terminal from notes panel

Try printing your current directory.

```bash
pwd
```
